O projeto Gestão Fácil foi desenvolvido exclusivamente para fins educativos, como parte de atividades acadêmicas do curso de Análise e Desenvolvimento de Sistemas. Todos os colaboradores participaram com o objetivo de aprendizado, sem qualquer intenção comercial ou lucrativa. Ressaltamos que a 'Gestão Fácil' é uma empresa fictícia, criada apenas como base conceitual para o desenvolvimento deste sistema.
